Sobre o Sistema

Sistema de Trilhas e Acampamentos

Este sistema foi desenvolvido para conectar trilheiros experientes com entusiastas da natureza, permitindo o compartilhamento de conhecimento sobre trilhas ecológicas, dicas de segurança e experiências de acampamento.

Funcionalidades Principais:
  • Cadastro e gerenciamento de trilhas
  • Sistema de avaliações e comentários
  • Diferentes níveis de usuário (Usuário, Trilheiro Experiente, Administrador)
  • Painel administrativo para gestão do sistema
  • Interface responsiva e intuitiva
Tecnologias Utilizadas:
  • Backend: Java 17, Spring Boot 3.x, Spring MVC
  • Frontend: Thymeleaf, Bootstrap 5, HTML5, CSS3
  • Build: Maven
  • Servidor: Tomcat Embarcado
